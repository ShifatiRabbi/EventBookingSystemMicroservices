To Run the Complete Test Suite:
Install load test dependencies:


cd scripts/load-test
npm install
Create test event:


curl -X POST http://localhost:3002/api/events \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Load Test Concert",
    "totalSeats": 1000,
    "date": "2024-12-31T20:00:00Z"
  }'
Run load test:


npm run test:load
Validate consistency:


npm run test:validation
Test failure scenarios:


# Stop Kafka temporarily
kubectl scale deployment kafka --replicas=0

# Make some bookings
curl -X POST http://localhost:3003/api/bookings \
  -H "Content-Type: application/json" \
  -d '{
    "eventId": "your-event-id",
    "userId": "test-user-1",
    "seatCount": 2
  }'

# Restart Kafka and check recovery
kubectl scale deployment kafka --replicas=1
